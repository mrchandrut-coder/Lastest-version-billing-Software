# ☕ Friends Tea Kadai — POS

A modern, mobile-first Point of Sale (POS) web application built for tea stalls, cafés, and small food businesses. Designed as a single HTML file, the application runs entirely in the browser without requiring a backend, database, or installation. All data is stored locally using the browser's Local Storage API, making it fast, lightweight, and offline-friendly.

**Live Demo:** https://mrchandrut-coder.github.io/Lastest-version-billing-Software/

## ✨ Features

### 🧾 Billing
- Quick item selection for fast order processing
- Increase or decrease item quantity
- Automatic bill total calculation

### 🔍 Menu Management
- Search menu items instantly
- Filter products by category
  - Tea
  - Snacks
  - Sweets
  - Other

### 💳 Payment Options
- Cash payments
- UPI payments
- Cash + UPI split payments
- Live validation for split payment totals
- Customer QR code payment support

### 📊 Reports & Analytics
- Today's sales summary
- Monthly sales overview
- Daily and weekly sales trend charts
- Complete transaction history

### 📁 Data Export
- Export sales history to CSV
- Compatible with Microsoft Excel and Google Sheets

### 🛠️ Item Management
- Add new products
- Edit existing products
- Delete menu items
- Customize product icons or images
- Set product prices

### 🔒 Security
- PIN-protected Settings
- PIN-protected Item Management

### 📱 Mobile Experience
- Responsive mobile-first interface
- Optimized for touch devices
- Supports Add to Home Screen for an app-like experience

## 🚀 Getting Started

### Run Online
Open the deployed GitHub Pages URL in any modern web browser.

### Run Locally
1. Download or clone this repository.
2. Open `index.html` in any modern browser.
3. Start using the application.

No installation, server, or dependencies are required.

### Deploy with GitHub Pages
1. Fork or clone the repository.
2. Navigate to **Settings → Pages**.
3. Select **Deploy from a branch**.
4. Choose the **main** branch and the **/ (root)** folder.
5. Save the configuration.


## 💾 Data Storage

This application stores all information locally in your browser using the Local Storage API, including:

- Menu items
- Sales history
- Payment settings
- Uploaded QR code
- Application preferences

No data is transmitted to external servers, and no user account is required.

**Note:** Clearing your browser's storage or cache will permanently remove the stored data.
